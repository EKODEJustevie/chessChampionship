Présentation du Tournoi

Découvrez le format innovant de compétition d'échecs en équipe

👥

Format Binôme

Chaque équipe est composée de 2 joueurs qui s'affrontent simultanément contre une équipe adverse. La stratégie d'équipe est primordiale pour la victoire.

⚔️

Confrontation Stratégique

Un joueur joue avec les Blancs et l'autre avec les Noirs. Le choix des couleurs avant le match est une décision tactique cruciale pour l'équipe.

🏆

Phases Finales

Les meilleures équipes de chaque groupe se qualifient pour les phases finales du vendredi. Qui remportera le titre de champion ?

🎯

Système de Points

3 points pour une victoire, 1 point chacun en cas de match nul. Chaque point compte pour la qualification aux phases finales.

Règles du Tournoi

Comprendre le fonctionnement des matchs et le système de victoire

1 Composition des Équipes

Chaque équipe est constituée de 2 joueurs. Avant chaque match, l'équipe doit décider :

  • Quel joueur jouera avec les Blancs
  • Quel joueur jouera avec les Noirs

Ce choix doit être communiqué à l'organisation avant le début du match.

2 Déroulement d'un Match

Deux parties simultanées sont jouées :

  • Partie 1 : Joueur A (Blancs) vs Joueur X (Noirs)
  • Partie 2 : Joueur B (Noirs) vs Joueur Y (Blancs)

Les résultats des deux parties déterminent l'issue du match.

3 Logique de Victoire

Le résultat du match se calcule selon les règles suivantes :

  • Victoire nette : Si les 2 membres d'une équipe gagnent leur partie → L'équipe gagne le match (3 pts)
  • Égalité 1-1 : Si un membre gagne et l'autre perd → Les 2 vainqueurs s'affrontent en départage. Le vainqueur du départage fait gagner son équipe (3 pts)
  • Double match nul : Si les 2 parties sont nulles → Les 2 vaincus s'affrontent en départage. Le vainqueur du départage fait gagner son équipe (3 pts)
  • Départage nul : Si le départage est aussi nul → Match considéré comme nul (1 pt chacun)

4 Système de Points

3 pts
Victoire
(1.5 pts × 2)
1 pt
Match Nul
(0.5 pts chacun)
0 pt
Défaite

5 Phases de Compétition

  • Jeudi (Phases de groupe) : Les équipes sont réparties en groupes. Chaque équipe affronte les autres de son groupe.
  • Vendredi (Phases finales) : Les meilleures équipes de chaque groupe se qualifient et s'affrontent en élimination directe.

Comment connecter les formulaires Google au site

📊 Structure des Google Sheets

1. Sheet ÉQUIPES (Inscriptions)

Formulaire d'inscription des équipes :

  • Colonne A : Horodatage
  • Colonne B : Nom de l'équipe
  • Colonne C : Joueur 1 (Nom complet)
  • Colonne D : Joueur 2 (Nom complet)
  • Colonne E : Groupe (A, B, C...)
  • Colonne F : Email de contact
https://docs.google.com/forms/d/e/[FORM_ID]/viewform

2. Sheet MATCHS (Résultats)

Formulaire de saisie des résultats :

  • Colonne A : ID Match
  • Colonne B : Équipe 1
  • Colonne C : Équipe 2
  • Colonne D : Score Équipe 1
  • Colonne E : Score Équipe 2
  • Colonne F : Résultat Échiquier 1
  • Colonne G : Résultat Échiquier 2
  • Colonne H : Date (JJ/MM/AAAA)
  • Colonne I : Heure (HH:MM)
  • Colonne J : Groupe
  • Colonne K : Statut (upcoming/live/finished)

3. Sheet JOUEURS (Stats)

Statistiques individuelles :

  • Colonne A : Nom du joueur
  • Colonne B : Équipe
  • Colonne C : Victoires
  • Colonne D : Défaites
  • Colonne E : Matchs nuls
  • Colonne F : Points totaux

4. Sheet CONFRONTATIONS (À venir)

Détails des matchs futurs :

  • Colonne A : ID Match
  • Colonne B : Joueur A (Équipe 1, Blanc)
  • Colonne C : Joueur B (Équipe 1, Noir)
  • Colonne D : Joueur X (Équipe 2, Blanc)
  • Colonne E : Joueur Y (Équipe 2, Noir)
  • Colonne F : Temps de jeu (ex: 15min+10s)
